# auto-tuic
安装 Sing-box 并通过菜单快速管理 TUIC / Reality 的自动化脚本。

## 使用

```shell
wget https://raw.githubusercontent.com/koopkl/auto-tuic/main/auto-tuic.sh && bash auto-tuic.sh
```

安装后按菜单选择功能即可。

## 当前菜单能力（简版）

- TUIC 安装与客户端配置导出
- Reality(VLESS+Reality) 一键安装
- sing-box 启停/重启/状态/日志
- Reality 客户端参数查看、URI 导出、二维码生成（依赖 qrencode）
